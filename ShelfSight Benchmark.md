# ShelfSight Benchmark.md
This file compares ShelfSight against common library systems to clarify product positioning and capstone targets.

## Feature Benchmark
| Feature | Koha | Follett Destiny | Libib | ShelfSight (Target) |
|---|---|---|---|---|
| Core catalog + circulation | Yes | Yes | Partial | Yes |
| Web-based admin workflows | Yes | Yes | Yes | Yes |
| AI-assisted metadata tagging | No (manual/plugins) | Limited | No | Yes |
| 2D shelf map visualization | No | No | No | Yes |
| First-person shelf view | No | No | No | Yes |
| Mis-shelved item detection support | Limited/manual | Limited/manual | No | Yes (AI-assisted) |
| Real-time shelf utilization analytics | Limited/reports | Limited/reports | Basic | Yes |

## Measurable Goals
- Reduce average new-book cataloging time by **30%** through AI pre-tagging.
- Achieve **95%+** shelf location accuracy in map-linked inventory records.
- Detect and flag **90%+** of intentionally misplaced test items during pilot scans.
- Keep dashboard and map interactions under **2 seconds** for standard librarian tasks.
- Reach **80%+** positive usability feedback from pilot staff surveys.
